# 🌿 Momentum: The Low-Energy Ecosystem

A high-performance sanctuary designed to combat executive dysfunction, isolation, and rejection. 

### 🛠 Technical Architecture
- **Framework:** Next.js 15 (App Router)
- **State/Theme:** React Context (Energy-Aware Vibe Engine)
- **Database/Auth:** Supabase (Real-time Pulses & Magic Links)
- **AI Engine:** Google Gemini (Task Deconstruction)
- **Visuals:** Framer Motion & Dynamic SVG (The Garden)

### 🚀 Core Systems
- **The Smasher:** AI-driven task deconstruction for "The Wall of Awful."
- **The Garden:** Real-time visual evolution based on Micro-Win logic.
- **The Worth Vault:** Encrypted repository for "Receipts of Competence" and artifacts.
- **The Anchor:** Somatic grounding & box-breathing utility.
